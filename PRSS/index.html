Take an exceptional set in the underlying ring R, of size n+1,
    {alpha0,...,alphan}.
This needs the Lenstra constant of the ring to be greater than or equal to n+1.

For a subset A ⊆ {1,...,n} of size n-t form the polynomial FA(X) of degree t such that

  • FA(alpha0)=1.
  • FA(alphai)=0 for i ∉ A.
Take PRF with codomain R and domain given by
    ℕ x ℕ x {0,1}κ
To each A ⊆ {1,...,n} of size n-t associate a key kA ∈ {0,1}κ and give kA to all parties in A.

Let S be a set of sets, each element A of S is a subset of {1,...,n} of size n-t. The set S needs to be rather special for the following to work. Below we give a table of the best (smallest) known sets for various parameters n and t.

Let cnt denote a counter.

To generate the cntth double sharing of degree (t,2t) of a random value r we compute the shares as

    (< r >t)i = ∑A ∈ S: i ∈ A PRF(cnt, 0, kA) × FA(i),
    (< r >2t)i = (< r >t)i + ∑A ∈ S: i ∈ A ( ∑j ∈ {1,...,t} PRF(cnt, j , kA) × ij × FA(i) ).
The value being shared is equal to
    r = ∑A ∈ S PRF(cnt, 0, kA) .

The traditional construction (CDI05) for the set S is to take all subsets of size n-t.

Thus the size of S in this traditional construction is binom(n,t), i.e. the size of the associated binomial coefficient.

However smaller sets S can be found which render the above PRSS construction of doubly-shared values secure.

For each entry in the table we give the size of S according to the traditional construction of CDI05, as well as the best set S we could find.
